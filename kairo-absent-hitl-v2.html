Reproduction Steps
  1. Open console and run Open L_ShooterGym?NumBots=0
  2. Optionally enable lyra.Weapon.DrawBulletTraceDuration 9999 and lyra.Weapon.DrawBulletHitDuration 9999 for visual debug.
  3. Equip the shotgun and fire once at a flat wall while standing still. Note the spread pattern....show more
Expected Behavior

When standing, crouching, or walking, the weapon spread should change according to the multiplier values set in the weapon data asset.

Actual Behavior

All movement states appear to use the same base spread value. Changing the per-state multipliers has no visible effect on the actual spread pattern.
